# Demo Payloads

Small demo payloads for manual testing.

- `minimal.json` is decoded schema JSON.
- `minimal.hsc1.txt` is the encoded payload string for Lovelace config.

The minimal demo includes basic primitives, a small generic `path` marker, two instances of the same reusable payload-defined symbol, an alarm badge controlled by `visibleWhen`, and a status dot controlled by `styleWhen`.

This is an early demo fixture, not a final editor export UI.
